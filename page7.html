1.7 Describe which are the appropriate filetypes for websites

HTML files are used to display information on websites. This can be plain text or can add in additional media through hyperlinks. CSS files are increasing popularity to style website with great flexibility and small file size. Photos and images can come in different file types and depending on what the image is for depends on what file type you probably want to use: JPG/JPEGs are used mainly for photos due to the compression rate, PNG files can use transparency however have a larger file type, PNGs are most commonly used for logos. GIFs care great for simple images or animated images. Music used on websites will commonly be a .mp3 or .wav file. Video content most commonly will be MP4, AVI or MPG/MPEG.
